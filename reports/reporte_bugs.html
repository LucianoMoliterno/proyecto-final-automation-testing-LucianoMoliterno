BUG-001 ALTA

Archivo > 5MB Aceptado sin Validación

Descripción:

El sistema acepta archivos de CV que exceden el límite de 5MB sin mostrar mensaje de error ni rechazar la carga. Esto puede generar problemas de almacenamiento y rendimiento.

Pasos para Reproducir:
  1. Ingresar a https://talentolab-test.netlify.app
  2. Hacer clic en "Registrate"
  3. Hacer clic en el campo de carga de CV
  4. Seleccionar un archivo > 5MB (ej: 6MB)
  5. Intentar enviar el formulario
Resultado Esperado:

El sistema debe rechazar el archivo y mostrar mensaje: "El archivo excede el límite de 5MB"

Resultado Obtenido:

El archivo es aceptado sin validación ni mensaje de error

Test Case Automatizado: test_tc009_carga_cv_limite_peso
Archivo: tests/test_carga_cv.py
BUG-002 MEDIA

Menú Hamburguesa No Responde en Móvil

Descripción:

En dispositivos móviles (resolución 2400x1080), el menú hamburguesa no responde al primer toque, requiere recarga de página. Además, el botón "Carga tu CV" presenta problemas visuales de espaciado.

Pasos para Reproducir:
  1. Abrir la web desde un dispositivo Android (2400x1080)
  2. Tocar el menú hamburguesa
  3. Observar que no se despliega
  4. Recargar la página
  5. Observar el botón "Carga tu CV"
Resultado Esperado:

El menú debe desplegarse al primer toque y todos los botones deben mantener espaciado correcto

Resultado Obtenido:

Menú no responde; requiere recarga. Botón "Carga tu CV" con espaciado incorrecto en móvil

Test Case Automatizado: test_tc010_navegacion_mobile
Archivo: tests/test_responsive.py
BUG-003 ALTA

Envío sin Validación de Campos Vacíos

Descripción:

El formulario de contacto permite el envío sin validar que los campos obligatorios estén completos. No muestra advertencias ni bloquea el envío cuando los campos están vacíos.

Pasos para Reproducir:
  1. Ir a la sección "Contacto"
  2. Dejar todos los campos vacíos (nombre, email, mensaje)
  3. Hacer clic en el botón "Enviar"
  4. Observar comportamiento
Resultado Esperado:

El formulario debe bloquear el envío y mostrar mensajes de validación en los campos obligatorios

Resultado Obtenido:

Los datos permanecen en el formulario sin validación ni mensaje de error

Test Cases Automatizados: test_tc003_envio_formulario_exitoso, test_tc004_formulario_campos_vacios
Archivo: tests/test_contacto.py
BUG-004 ALTA

Link "Carga tu CV" Devuelve 404

Descripción:

El link del botón "Carga tu CV" en el menú principal redirige a una página que devuelve error 404 - Not Found. Esto impide que los candidatos puedan cargar su CV.

Pasos para Reproducir:
  1. Ingresar al sitio web
  2. Hacer clic en "Carga tu CV" en el header
  3. Observar la página resultante
Resultado Esperado:

Redirección a formulario de carga de archivo o página de login funcional

Resultado Obtenido:

Página 404 - Not Found (https://talentolab-test.netlify.app/login no existe)

Test Case Automatizado: test_tc002_carga_cv_valido
Archivo: tests/test_carga_cv.py
Nota: Test documenta el error 404 actual
BUG-005 MEDIA

Archivo Corrupto No Validado

Descripción:

El sistema no valida la integridad de los archivos cargados como CV. Acepta archivos corruptos o con formato inválido sin mostrar advertencia.

Pasos para Reproducir:
  1. Ir a la página de registro
  2. Completar los campos del formulario
  3. Seleccionar un archivo corrupto o inválido (.docx dañado)
  4. Intentar cargar el archivo
Resultado Esperado:

Rechazo inmediato con mensaje: "El archivo está corrupto o tiene formato inválido"

Resultado Obtenido:

La plataforma procesa el CV corrupto sin mostrar advertencia ni bloqueo

Test Case Automatizado: test_tc008_carga_cv_corrupto
Archivo: tests/test_carga_cv.py
